06 -- Analytical Methods

Analytical Toolkit

Adapted from our West Africa infrastructure resilience project. The same graph-theoretic methods applied to Lebanon's grid topology.

∧
Articulation Points

Substations whose failure fragments the grid into disconnected components. These are the single points of failure that defense strategy must protect.

✂
Min-Cut Analysis

For each load zone, the minimum set of transmission lines whose destruction would isolate it. Shows structural vulnerability independent of generation capacity.

⇄
Route Redundancy

How many alternative paths exist from each plant to each substation. Low redundancy indicates fragile connections vulnerable to single-line disruption.

■
Community Detection

Natural grid clusters that form quasi-independent sub-grids. These communities reveal which regions could operate autonomously during partial grid failure.

Δ
Opportunity Signals

Gap between structural capacity (what the topology supports) and actual throughput. Identifies underutilized infrastructure and bottlenecks limiting power delivery.

↻
Cascade Simulation

When a line exceeds capacity, it trips -- redistributing load to remaining paths. If those paths also overload, the cascade continues until equilibrium or blackout.
